# Expense Tracker

This is a simple command-line tool written in Rust to track your expenses. It helps you add expenses, view reports, and export your data to CSV files.

## Features
- Add expenses to different categories.
- View a report of your expenses.
- Export your expenses to a CSV file.

## Usage

After downloading the binary, you can use it like this:
./expense_tracker --add 10 "food"

This will add a $10 expense under the "food" category.

## Installation

You can download the latest version of the tool from the **Releases** section of this repository.

## Build (GitHub Actions)

The project is automatically built using GitHub Actions. You can find the Linux binary in the **Releases** section.
